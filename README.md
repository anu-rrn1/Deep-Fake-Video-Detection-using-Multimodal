Description of this project:
Developed a multimodal deepfake detection framework using PyTorch, OpenCV, Librosa, Torchaudio, and FFmpeg, leveraging ResNet-18 for visual artifact detection, MFCC-based feature extraction with statistical audio sampling, Bi-LSTM + 1D CNN for audio temporal modeling, and cross-modal attention fusion for audio-visual correlation learning. Achieved 83.47% Accuracy, 83.06% AUC, and 85.01% F1-Score on 23K+ FakeAVCeleb v1.2 videos.

Tech Stack: PyTorch, Scikit-learn, OpenCV, Librosa, Torchaudio, and FFmpeg
